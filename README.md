2. Open BRAT settings (Settings â†’ BRAT)
3. Click **Add Beta Plugin**

5. Click **Add Plugin**

**Benefits**: Automatic updates, smooth experience, one-click installation

   - `styles.css`
3. Create folder: `{VaultFolder}/.obsidian/plugins/bases-improvements/`
